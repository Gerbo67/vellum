# Configuración {#top}

Las opciones de configuración están en el menú del icono de engranaje, en la esquina inferior del sidebar, junto al avatar del usuario. La configuración se guarda por usuario y persiste entre sesiones.

## Notificaciones {#notifications}

Activa las notificaciones de escritorio para recibir un aviso cada vez que Vellum recibe un correo nuevo, incluso si la pestaña del navegador está en segundo plano.

Para habilitarlas, el navegador pedirá permiso la primera vez que pulses la opción. Si el sistema operativo o el navegador bloquean los permisos, la opción aparecerá como no disponible.

- **Bloqueadas en el navegador** — el permiso fue denegado por el usuario o bloqueado por la configuración del sitio. Para reactivarlas hay que cambiar el permiso manualmente en la configuración del navegador.
- **No soportadas** — el navegador no implementa la API de notificaciones. Ocurre en algunos navegadores móviles o entornos sin soporte.

Las notificaciones muestran el nombre del proyecto y el asunto del correo recibido.

## Apariencia {#appearance}

Controla el esquema de color de la interfaz de Vellum.

### Claro {#theme-light}

La interfaz usa siempre el tema claro, independientemente de la configuración del sistema operativo o del navegador.

### Sistema {#theme-system}

La interfaz sigue la preferencia de color del sistema operativo. Si el sistema está en modo claro, Vellum se muestra en claro. Si está en modo oscuro, Vellum se adapta automáticamente. Esta es la opción predeterminada.

### Oscuro {#theme-dark}

La interfaz usa siempre el tema oscuro, independientemente de la configuración del sistema.

## Idioma {#language}

Cambia el idioma de la interfaz entre español e inglés. El cambio aplica de forma inmediata sin necesidad de recargar la página.

El idioma también afecta al análisis de correos: los nombres de las verificaciones, los detalles de cada check y el resumen de puntuación se devuelven en el idioma seleccionado.
